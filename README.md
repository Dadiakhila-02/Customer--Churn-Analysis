📊 Customer Churn Analysis Dashboard (Power BI)

Customer churn is one of the most critical challenges faced by businesses today. Retaining existing customers is often more cost-effective than acquiring new ones. This project focuses on analyzing customer data to identify patterns, trends, and key factors that contribute to customer churn.

Using Power BI, the dashboard provides interactive visualizations that help organizations understand customer behavior, monitor churn rates, and make data-driven decisions to improve customer retention.

🎯 Project Objectives

Analyze customer data to identify churn patterns.

Understand the key factors influencing customer churn.

Visualize important metrics such as churn rate, customer demographics, and service usage.Provide insights that help businesses reduce customer churn and improve retention strategies.

🛠️ Tools & Technologies Used

Power BI – Data visualization and dashboard creation

Microsoft Excel / CSV – Data source and preprocessing

Data Cleaning & Transformation – Power Query

Data Modeling – Relationships and calculated measures using DAX.
📂 Dataset Description

The dataset contains customer information including:

Customer ID

Gender

Age / Demographics

Contract Type

Tenure

Internet Services

Payment Method

Monthly Charges

Total Charges

Churn Status (Yes/No)

This data helps analyze how different customer attributes influence churn behavior.

📌 Conclusion

This project analyzes customer data to identify patterns and key factors influencing customer churn. The Power BI dashboard provides clear insights into customer behavior, helping businesses understand why customers leave. These insights can support better decision-making and help organizations improve customer retention strategies.
